# Chess Engine

Simple chess engine written in C++.

Features:

- Bitboard representation
- Move generation
- Alpha-Beta search
- MVV-LVA move ordering
- UCI protocol support

## Build

Requires:

- C++20
- g++

Compile:

g++ -std=c++20 Engine.cpp ...

## Future improvements

- Quiescence search
- Transposition tables
- Iterative deepening
